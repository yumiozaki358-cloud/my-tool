# My Tool

シンプルなランディングページです。

## 概要

作業を効率化するツール「My Tool」の紹介ページです。Hero・Features・CTA の3セクションで構成されています。

## ファイル構成

```
.
├── index.html   # ページ本体
└── style.css    # スタイル
```

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバー不要です。

## 機能

- **Hero セクション** — キャッチコピーと CTA ボタン
- **Features セクション** — 3つの特徴カード（高速処理・安全・シンプル設計）
- **CTA セクション** — メールアドレス登録フォーム

## 技術スタック

- HTML5
- CSS3（CSS Variables・Grid・Flexbox）
